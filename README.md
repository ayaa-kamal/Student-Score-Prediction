# Student Exam Score Prediction

## Project Overview

This project aims to predict student exam scores using Machine Learning techniques.

## Dataset

The dataset contains information about students such as:

- Hours Studied
- Attendance
- Sleep Hours
- Previous Scores
- Tutoring Sessions
- Physical Activity

Target Variable:

- Exam Score

## Models Used

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

## Results

| Model | R² Score |
|---------|---------|
| Linear Regression | 0.77 |
| Random Forest | 0.65 |
| Decision Tree | -0.04 |

## Best Model

Linear Regression achieved the best performance.
